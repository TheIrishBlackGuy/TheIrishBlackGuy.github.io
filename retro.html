The Retrobox

With this project, I set out to make a small interactive 'game'. While working on this, I came to realize that even simple games require more game design than I had thought. I'd like to think I succeeded in making a frame to potentially contain a game but that is all I think I did. Sure you can move the character but that is just interaction, not a game. As I continued this, I began to think of this as more of a proof of concept and practice towards making an interac tive setup with a goal. This is not finished nor will it be finished for a lenght of time. But there is a goal to be aimed for now.
